  • Tentang Send Whatsapp
    Send Whatsapp Memudahkan anda berkirim Pesan Whatsapp Tanpa menyimpan Kontak nomor telepon yang ingin anda hubungi
    Fitur:

    1. Send Whatsapp- Anda dapat mengirim pesan dan media langsung di WhatsApp tanpa menyimpan kontak

    2. Tidak perlu menyimpan Kontak - Tidak perlu menyimpan nomor di Kontak, langsung tambahkan / pilih nomor dan Kirim

    3. Pilih nomor secara langsung dari Log panggilan - Cukup pilih nomor dari log panggilan terbaru dan kirim pesan

    4. Mudah mengirim pesan (Cara kerjanya)
    1. Pilih nomor WhatsApp
    2. Masukkan pesan teks
    3. Ketuk “KIRIM” untuk mengirim pesan
    4. Anda dapat melampirkan media dari tombol lampiran aplikasi WhatsApp.

    catatan:

    - Aplikasi ini membutuhkan persyaratan kebijakan untuk penggunaan yang dapat diterima atau memenuhi syarat untuk pengecualian, Anda harus mendeklarasikan Log Panggilan atau izin SMS langsung melalui Play Console.

    - Aplikasi ini menggunakan API whatsapp resmi untuk membuka obrolan dengan nomor apa pun yang Anda masukkan tanpa membuatnya sebagai kontak di perangkat Anda.

    - Aplikasi atau pengembang aplikasi ini tidak terkait dengan WhatsApp. WhatsApp adalah merek dagang terdaftar dari WhatsApp Inc.

    - Anda harus mengikuti syarat & ketentuan WhatsApp saat mengirim pesan melalui Send Whatsapp.
